# mifa.jekyll

Mifa GitHub Pages 主题

Demo 地址：[https://phodal.github.io/mifa-jekyll/](https://phodal.github.io/mifa-jekyll/)

使用方式，在 GitHub Pages 里创建 ``_config.yml`` 文件，并添加以下的内容来启用 Mifa 主题：

```
remote_theme: phodal/mifa-jekyll
```
